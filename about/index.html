# 博客文章管理

# 序，先决条件

  • 首先需要安装 git 软件，配置 git 的 ssh 密钥，配置全局邮箱和用户名；
  • 将项目文件克隆到本地；
  • 安装 node 软件（版本要高于 10），安装 hexo；
  • 接下来就可以编写文章，并上传了到 github 了。
